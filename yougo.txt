1章
1. コンピューターとは
・電子計算機(電子(電気信号)で動作する計算目的の機械)
電気信号＝＞電圧を上下することで0と1の2種類の数値を表現
・0と1の2種類の組み合わせで数値を表現＝＞2進数、デジタル
・アナログ＝＞情報を連続的な変化によって表す

2.5大装置
・入力装置:人間の命令やデータをコンピュータに入力する
＝＞キーボード、マウス、スキャナー、マイク

・記憶装置:命令やデータを一時的または恒久的に記憶する
＝＞メモリー、ハードディスク、光学式メディア、フラッシュメモリ
ー＞主記憶装置(メインメモリ):データを一時的に保存
コンピュータの電源が入っている間だけ記憶を保存するRAM(Random Access Memory)
ー＞補助記憶装置:データを長期的に保存
DVD(Digital Versatile Disc),SSD(Solid State Drive)

・制御装置:入力装置、記憶装置、演算装置、出力装置を制御し、命令やデータを各装置に伝える
＝＞CPU(Center Processing Unit:制御演算装置)、チップセット

・演算装置:命令やデータに従い四則演算や論理演算などの計算を行う
＝＞CPU

・出力装置:計算した結果を命令に従って人間にわかる形で出力する
＝＞ディスプレイ、プリンター、スピーカー

コンピュータの内部では5大装置がマザーボードに接続される
マザーボードの設定や制御はマザーボード上のBIOS(Basic Input/Output System)で行われる

3.コンピューターの種類
・クライアント＝＞サービスの提供をうける
・サーバー　＝＞サービスを提供する
サーバーは提供するサービスによって様々な名称で呼ばれている
ー＞メールサーバー:メールの送受信
ー＞Webサーバー:Webページの提供
ー＞ファイルサーバー:ファイル管理
ー＞プリントサーバー:印刷

大きさによる分類
PC(Personal Computer)、モバイルデバイス(スマートフォン、タブレット)

大  ラックマウントPC
    タワーPC
    デスクトップPC
    ノートPC
    タブレット
小  スマートフォン

・システムソフトウェア
ー＞OS(Operating System)、ハードディスクを制御するデバイスソフトウェア、OSとアプリケーションの中間で、共通の処理を行うミドルウェア

・アプリケーション
ー＞ユーザーが日常業務など特定の目的などに使用される

2章
1.マザーボードとは

・マザーボード
ー＞コンピューター内部の各部品を相互接続する板状の基盤

マザーボードの構成要素
・CPUソケット
ー＞コンピュータの頭脳プロセッサ(CPU)を接続する場所

・メモリソケット
ー＞データを記憶しておくメモリモジュールを接続する場所

・各種コネクタ
ー＞電源ケーブルやストレージ(補助記憶装置)であるハードディスクやCD-ROMなどの接続ケーブルをつなぐための接続口IDE(Integrated Drive Electoronics)やSATA
(Serial Advanced Technology Attachment)などの規格がある

・拡張スロット
ー＞様々な機能を実現するための拡張カードと呼ばれる部品を差し込むためのスロットPCI(Peripheral Component Interconnect)やPCI-Expressなどの規格がある

・バス
ー＞マザーボードに接続された各装置の間をつなぎ信号をやり取りする伝送路

・その他
ー＞回路群(チップセット)、BIOS

フォームファクター
ー＞マザーボードの規格。ATX(Advanced Technology eXtended)

2.PCの電源

・電源ユニット
ー＞PSU(Power Supply Unit)コンピューターに電力を供給する機器。

・電圧　ー＞電気を流そうとする力単位はV(Volt)
・所要電力　ー＞ハードウェアが動作に必要な電力単位はW(Watt)またはVA(Volt-Ampare)
・各種電源ユニットー＞マザーボードへ給電するための接続コネクタの他にCPUやメモリ、ハードディスクなどへ給電する専用の電源コネクタ形状が定められている
SCSI(Small Computer System Interface)接続のハードディスク
ー＞コンピューターの電源をいれる前に接続しなければ認識されないデバイス
USB(Universal Serial Bus)
ー＞コンピューターの電源が入った状態でデバイスの切り離しが可能な仕組みホットスワップに対応。コンピューター起動後に接続しても問題なく認識される

3.プロセッサ

・プロセッサ
＝＞コンピューターの頭脳に当たる部分。各種ハードウェアを制御し、四則演算や論理演算を行う。CPU(Center Processing Unit),中央処理装置

・プロセッサの性能
クロック周波数
ー＞コンピューターが動作する際に内部装置が同期をとるための周期的な信号
単位はHzプロセッサが1秒間に命令を処理する回数を示しているので、数値が大きいほどプロセッサの性能が高い
k(キロ) 1,000
M(メガ) 1,000,000
G(ギガ) 1,000,000,000
T(テラ) 1,000,000,000,000

コア数
ー＞主な演算や制御を実行するための中核となる回路をコアという。内蔵するコア数画多いほどプロセッサの性能が向上する。
マルチコアプロセッサ＝＞1つのプロセッサの中に複数のコアを搭載したプロセッサ
シングルコアプロセッサ＝＞コアが1つ

32/64ビット対応
ー＞プロセッサが一度に扱える情報量。大きさが64ビット分(0、1だけを使う2進数の64桁分)は64ビットCPU、32ビット分なら32ビットCPUと呼ばれる

CPUキャッシュ
ー＞メインメモリから送られた命令を処理する際メインメモリの動作が低速なため、速度差の影響を緩和するため、以前使った使用頻度の高いデータを一時的に保持する、メインメモリより高速なメモリ。

プロセッサの種類

64ビット/CPU:Core i7 Extreme/コア数:8,6/用途:ワークステーション
64ビット/CPU:Core i7/コア数:4/用途:デスクトップPCまたはノートPC
64ビット/CPU:Core i5/コア数:4,2/用途:デスクトップPCまたはノートPC
64ビット/CPU:Core i3/コア数:2/用途:デスクトップPCまたはノートPC
64ビット/CPU:Celeron/コア数:2,1/用途:デスクトップPCまたはノートPC
64ビット/CPU:Pentium/コア数:2/用途:デスクトップPCまたはノートPC

-AWD社製プロセッサ-
64ビット/CPU:Opteron/コア数:16,12,8,6,4/用途:ワークステーション
64ビット/CPU:AMD FX/コア数:8,6,4/用途:デスクトップPC
64ビット/CPU:AMD APU/コア数:4,2/用途:デスクトップPCまたはノートPC

GPU
ー＞GPU(Graphics Processing Unit)画像処理を行う半導体チップ。オンボードで提供されているものと拡張カードとして追加するものがある。ディスプレイに表示するデータを保持するVRAM(Video Random Access Memory)などと連携することによってより多くの画像処理を行うようになった。

冷却装置
ー＞熱暴走を起こさないようにする装置
・空冷式
＝＞一般的に用いられる。ヒートシンクとファンを使用
ヒートシンク:プロセッサに取り付ける放熱板。熱伝導率の高いアルミや銅製
CPUファン:ヒートシンクに空気を送り冷却する装置
ケースファン:PC内部に空気の流れを作り部品を冷却する

・液体冷却
ー＞水冷式。液体を循環させることでプロセッサを冷却する。大型のサーバーなどで使用される

4.メモリ

コンピューターの処理を本を広げて作業することに例えると……
プログラム、データ　＝＞本
メインメモリ　＝＞本を広げる作業机
補助記憶装置(ストレージ)　＝＞本棚

メモリの種類
RAM
ー＞(Random Access Memory)揮発性のメモリ。コンピューターの電源が切れるとデータも消える。メインメモリ、キャッシュメモリ
ROM
ー＞(Read Only Memory)不揮発性。電源が切れてもデータを保持する。ROM、BIOS

メモリモジュール
DIMM(Dual Inline Memory Module):デスクトップPCで広く使用されている
S.O.DIMM(Small Outline DIMM):DIMMを小型化したモジュール。ノート型PCで広く使用されている。SO-DIMM

メモリの性能
ー＞サイズ(容量)が大きいほど一度に扱える命令やデータの数が増える。動作周波数の数値が大きいほどデータ転送速度も高くなる。

5.ハードディスク

ハードディスク
ー＞磁気ディスクを利用してデータを読み書きする補助記憶装置(ストレージ)
構造:磁気ディスクが密閉されたケースに入っており高速回転している。アームの先にある磁気ヘッドを使って、磁気ディスクにデータの読み書きを行う

構造上の問題点
ー＞衝撃や落下に弱い。消費電力が大きく、低電力化が難しい。振動が発生し音がうるさい。読み書きの高速化が難しい。
解消するために………
ー＞フラッシュメモリを用いたSSD(Solid State Drive)が使用されることが多い

ハードディスクの種類
内蔵:コンピューター内部に接続。場所を取らず、転送速度が外付けより速い
外付け:コンピューター外部に接続。別のコンピューターにも接続できる。

ハードディスク接続インターフェイス
IDE
ー＞IDE(Integrated Drive Electoronics)、ATA(Advanced Technology Attachment)とも呼ばれ1つのインターフェイスにハードディスクを2台まで接続できる。後に規格が拡張され、EIDE(Enhanced IDE)となり、2つのインターフェイスにハードディスクまたはCD-ROMを4台まで接続できるようになった。

SATA
ー＞SATA(Serial ATA)IDEより高速なデータ転送が可能で、デスクトップPCだけでなくノートPCでも主流の内蔵インターフェイス。一つのコネクタに1台接続できる。外付け用に拡張されたeSATA(External SATA)もある

SCSI/SAS
ー＞SCSI(Small Computer System Interface)サーバーの内蔵または外付けのハードディスクやバックアップ装置の接続に広く使用されていたインターフェイス。最近ではより高速なSAS(Serial Attached SCSI)が使用されている

ハードディスクのサイズ
デスクトップPC:3.5インチ、2.5インチ
モバイルノートPC:1.8インチ

ハードディスクの性能
rpm
ー＞rpm(Revolution Per Minute)ハードディスクの1分間の回転数を表す単位。値が大きいほど高速に処理できる。

シークタイム
ー＞データを読み取るための磁気ヘッド部分とアームを適切な位置に動かすための時間。短いほど高性能。

データ転送速度
IDE(ATA133)/133MB/s
SATA(SATA3.0)/600MB/s
SCSI(Ultra320)/320MB/s

キャッシュ(ディスクキャッシュ)
ー＞プロセッサと同様、使用頻度の高いデータをキャッシュとして保管することで、高速に読み書きを行うことが出来る。キャッシュ容量が大きいほどデータ処理の遅延の可能性を軽減することができる。

現在数百Gバイト〜数Tバイトの容量のものが多く用いられている。

その他のストレージ
・SSD
ー＞半導体メモリの一種。物理的な駆動部がないフラッシュメモリを使用する。そのため、ハードディスクよりも高速な書き込みが可能、衝撃にも強い。高価。

・光学式ドライブ
ー＞記憶媒体にレーザー照射などを使用することでデータの読み書きを行う補助記憶装置。CD(Compact Disc),DVD(Digital Versatile Disc),Blu-rayなどの規格がある。

6.拡張カード

拡張カード
ー＞コンピューター内部に機能を追加したり外付け用インターフェイスを増設するための基板

・拡張カードの種類
ビデオカード
ー＞グラフィックカード。コンピューターに映像を信号として入出力するための拡張カード。
内部インターフェイス:コンピューターにビデオカードを接続するインターフェイス。かつてはAGP、最近はPCI Expressが使用されている
VRAM:(Video RAM)表示する描画情報を保持するためのバッファ領域。
外部インターフェイス:ビデオカードの出力をディスプレイへ接続するためのインターフェイス。かつてはVGA(Video Graphics Array)、現在はDVI(Digital Visual Interface)やHDMI(HIgh-Definition Multimedia Interface)が使用されている。

オーディオカード
ー＞サウンドカード。コンピューターに音響を信号として入出力するための拡張カード。

ネットワークカート
ー＞LANカードやNIC(Network Interface Card)とも呼ばれる、コンピューターをネットワークへ接続するための拡張カード。

モデム
ー＞電話回線を使用してデータのやり取りを行うために、デジタル信号をアナログ信号に変調して送信したり、アナログ信号をデジタル信号に復調して受信する装置。
